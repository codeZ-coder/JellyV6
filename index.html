Manifesto

Uma água-viva é 98% água. Sem cérebro, sem ossos, sem plano fixo. Só o ambiente, a corrente e a resposta ao que encontra pelo caminho.

Jelly é um experimento com a mesma lógica — um sistema que aprende com o que encontra, não com o que foi programado para esperar.

"Não construo para durar.
Construo para evoluir."

Cada repositório aqui é uma célula tentando aprender a brilhar. O processo é público porque o erro também ensina — especialmente o erro de quem está começando do zero, aos 40+, sem manual.

Jelly não é um produto. É um log de transição — o diário de um sistema que está se descobrindo em tempo real.

Princípios
⌇

Flutuar

Não preciso saber tudo. Sinto o ambiente, me adapto, sigo. A corrente decide tanto quanto eu.

◈

Brilhar

Não para chamar atenção — para iluminar o caminho de quem vem depois. Aprendizado público é bioluminescência.

∞

Pulsar

Contrair, expandir, seguir. Sem órgãos, só intenção. O ritmo é o método.
